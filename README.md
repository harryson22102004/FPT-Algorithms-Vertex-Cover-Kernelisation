# FPT Algorithms: Vertex Cover & Kernelisation
Efficiently solving NP-Hard problems using Parameterized Complexity.

### 🔬 Research Highlights
- **Kernelisation**: Implements pre-processing rules to reduce a large graph to a small "kernel" whose size depends only on the parameter $k$.
- **Bounded Search Trees**: Uses recursive branching to find a vertex cover in $O(2^k \cdot poly(n))$ time.
- **Algorithms Focus**: Proves that hard problems can be solved efficiently if the "hardness parameter" is small.

**Tech Stack:** Python, `NetworkX`.
